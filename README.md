ATM Simulation System 

The ATM Simulation System is a console-based banking application developed in C that replicates the core functionalities of an Automated Teller Machine (ATM). This project demonstrates how fundamental programming concepts can be applied to simulate real-world banking operations such as user authentication, cash withdrawal, deposits, and balance inquiries.

Features

- Secure user authentication using account number and PIN
- Cash withdrawal functionality with balance validation
- Deposit feature to update account balance
- Real-time balance checking
- Interactive menu-driven user interface

Technologies Used

- C Programming Language
- Standard C Libraries
- Structured Programming Concepts

System Workflow

The program begins by authenticating the user through a predefined account number and PIN. Once successfully authenticated, the user is presented with a menu that allows them to perform banking operations such as withdrawing funds, depositing money, and checking the account balance. The system continuously updates and manages the account balance based on user actions.

Key Learning Outcomes

- Applied core C programming concepts including loops, functions, and conditional statements
- Designed a menu-driven interactive console application
- Simulated real-world banking transaction workflows
- Strengthened logical thinking and problem-solving skills

How to Run the Project

Step 1: Compile the Program

gcc atm.c -o atm

Step 2: Execute the Program

./atm

Project Objective

The objective of this project is to demonstrate the implementation of a simplified ATM system while reinforcing programming fundamentals and algorithmic thinking using the C programming language.
